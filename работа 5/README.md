# Работа 5 

Необходимые файлы:
- `work5_run.py` — генерация данных и вызов Prolog,
- `prolog_maximal_coalitions.pl` — Prolog-алгоритм,
- `games_data.py` — локальная база 50 игр,
- `run.bat` — запуск,


## Что происходит при запуске
1. Берет игры из локального файла `games_data.py`.
2. Генерирует MATCHES_COUNT партий и PLAYERS_COUNT игроков случайно.
3. В каждой партии выбирает случайную игру и число участников в диапазоне `min..max`.
4. Строит граф встреч игроков.
5. Prolog находит все максимальные группы, где каждый играл с каждым.
6. Сохраняет полный отчет в `report.txt`:
   - количество партий/игр/игроков,
   - топ игрока и игры по числу партий,
   - все максимальные коалиции (сортировка по убыванию размера),
   - список всех игр,
   - список всех игроков.

## Как запустить
- `run.bat`.
- Либо в терминале: `python work5_run.py`.

## Настройка количества
В `work5_run.py`:
- `PLAYERS_COUNT` — число игроков,
- `MATCHES_COUNT` — число партий,
- `GAMES_COUNT` — число игр в выборке.
- `SEED` - меняет выборку
