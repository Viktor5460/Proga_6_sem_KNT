## Состав работы

### Задача 1Б (файл `1b.hs`)
**Условие:** бесконечный список частичных сумм гармонического ряда, где элемент с номером `n` равен:
`f_n = sum (1 / i), i = 1..n`.

**Реализация:** функция `harmonicPartialSums :: [Double]`.

Пример в `ghci`:
- `take 6 harmonicPartialSums`
- результат: `[1.0,1.5,1.8333333333333333,2.083333333333333,2.283333333333333,2.45]`

### Задача 2А (файл `2a.hs`)
**Условие:** бесконечный список всех двумерных точек с целочисленными координатами.

**Реализация:** функция `integerPoints2D :: [(Integer, Integer)]`.

Пример в `ghci`:
- `take 12 integerPoints2D`

### Задача 3А (файл `3a.hs`)
**Условие:** бесконечный список всех приближений обыкновенной дроби `num/den`,
где `num` и `den` задаются параметрами.

**Реализация:** функция `fractionApproximations :: Integer -> Integer -> [Double]`.

Пример в `ghci`:
- `take 6 (fractionApproximations 2 3)`
- результат: `[0.7,0.67,0.667,0.6667,0.66667,0.666667]`

### Задача 4А (файл `4a.hs`)
**Условие:** тип данных «число, разложенное на простые множители» с операциями
быстрого поиска НОД и НОК.

**Реализация:**
- `data PrimeFactorNumber`
- `fromIntegerPF`, `toIntegerPF`
- `gcdPF`, `lcmPF`

Пример в `ghci`:
- `let a = fromIntegerPF 360`
- `let b = fromIntegerPF 840`
- `toIntegerPF (gcdPF a b)`  -- `120`
- `toIntegerPF (lcmPF a b)`  -- `2520`

### Задача 5А (файл `5a.hs`)
**Условие:** тип данных «конечное поле заданного константой порядка `q`» с операциями
сложения, умножения, взятия противоположного и обратного элементов.

**Реализация:**
- `data FiniteField`
- `mkField`, `addF`, `mulF`, `negF`, `invF`

Пример в `ghci`:
- `let x = mkField 7 3`
- `let y = mkField 7 5`
- `addF x y`   -- `1 (mod 7)`
- `mulF x y`   -- `1 (mod 7)`
- `negF x`     -- `4 (mod 7)`
- `invF x`     -- `5 (mod 7)`
