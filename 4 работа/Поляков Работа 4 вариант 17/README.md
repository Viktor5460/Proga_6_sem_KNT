# Обработка списков на Prolog (Работа 4)

В этой папке находятся решения лабораторной работы №4 по Prolog.

Варианты по таблице:
- `1Б`
- `2А`
- `3А`
- `4А`
- `5А`


### Задача 1Б (файл `1b.pl`)
**Условие:** найти всех, у кого есть дети от двух разных людей.

### Задача 2А (файл `2a.pl`)
**Условие:** удалить каждый третий элемент исходного списка.

### Задача 3А (файл `3a.pl`)
**Условие:** проверить, что длина списка чётная.

### Задача 4А (файл `4a.pl`)
**Условие:** проверить, является ли первый список окончанием второго.

### Задача 5А (файл `5a.pl`)
**Условие:** дан список, элементы которого тоже списки. Сформировать один список из их содержимого.
