# BoardGame Advisor - Приложение

## Структура файлов

Для работы приложения нужны следующие файлы:

1. **board_game_lib.dll** - скомпилированная библиотека C++
2. **boardgame_lib.py** - Python обёртка для работы с DLL
3. **app_gui.py** - главное GUI приложение
4. **default_games.py** - модуль для загрузки игр по умолчанию (опционально)
5. **settings.json** - файл настроек (создаётся автоматически)

## Запуск приложения

### Вариант 1: Через bat-файл
Дважды кликните на `run.bat`

### Вариант 2: Через командную строку
```bash
python app_gui.py
```

### Вариант 3: Прямой запуск Python файла
Дважды кликните на `app_gui.py` (если Python ассоциирован с .py файлами)

## Требования

- Python 3.7 или выше
- Установленный tkinter (обычно идёт в комплекте с Python)
- Windows (для работы с .dll)

## Возможные проблемы

### Ошибка "Не удаётся найти board_game_lib.dll"
- Убедитесь, что файл `board_game_lib.dll` находится в той же папке, что и `app_gui.py`
- Проверьте, что имя файла точно `board_game_lib.dll` (с учётом регистра)

### Ошибка "No module named 'default_games'"
- Скопируйте файл `default_games.py` в папку с приложением
- Или запустите приложение с опцией "Создать новую вселенную" (без игр по умолчанию)

### Ошибка импорта tkinter
- Установите Python с поддержкой tkinter
- Или установите отдельно: `pip install tk`

## Примечание

C++ исходники не нужны для работы приложения - только скомпилированная DLL.
